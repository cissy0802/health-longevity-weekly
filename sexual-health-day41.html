健康长寿：性健康与功能
Sexual Health & Function — Hormones, Pelvic Floor, Vessels & Bonds

本期证据等级：以大型队列 + RCT/meta 为主；激素与盆底建议综合内分泌学会、Cochrane、NAMS 与 Attia/Haver 指南
SUB · 性激素 / 内分泌
队列 + RCT
性激素与年龄：缓降 vs 断崖
Sex Hormones & Aging — Gradual Slope vs Cliff
一句话结论
男性睾酮 30 岁后每年约降 1%（缓坡），女性雌激素在围绝经期 断崖式下跌。补充激素是对有症状 + 实验室确诊缺乏者的治疗，不是人人适用的抗衰神药。
科学背景 + 机制
性激素调控肌肉、骨密度、性欲、情绪、糖脂代谢与血管内皮。男性睾酮平缓下行，多数仍在正常区间；真性性腺功能减退需两次清晨总睾酮 < 264 ng/dL（约 9.2 nmol/L）+ 症状方能诊断。女性则在末次月经前后数年雌二醇剧烈波动后骤降，触发血管舒缩症状、骨流失与泌尿生殖萎缩。TRAVERSE 试验（2023, NEJM, n=5246）证实：对确诊性腺减退男性，规范睾酮治疗未增加主要心血管事件，纠正了长期安全争议。
可执行协议
• 男性查什么：清晨（8–10 时）总睾酮 + 游离睾酮 + SHBG + LH；低值需复查第二次再判读。
• 女性查什么：围绝经期诊断靠症状 + 月经史，FSH 单点意义有限；不靠"测激素"决定。
• 治疗门槛：男性 TRT 仅用于确诊性腺减退（睾酮低 + 症状），并监测红细胞压积、PSA。
• 先改生活方式：睡眠、减脂、抗阻训练、控酒可显著抬升内源睾酮，常无需用药。
女性特别提示 + 常见误区
女性视角：围绝经期激素治疗（MHT）的"时间窗假说"——60 岁前或绝经 10 年内启动，获益（控热潮、护骨、缓泌尿生殖症状）大于风险（NAMS 2022 立场声明）。2002 WHI 的恐慌源于平均 63 岁、晚启动人群，已被重新解读。睾酮在女性主要用于性欲低下（HSDD），属适应证外但有证据。
误区 1：睾酮 = 抗衰万能药 → 对睾酮正常者无获益，反致睾丸萎缩、不育、红细胞增多。
误区 2：补睾酮致前列腺癌 → 大型数据未支持因果（"饱和模型"）。
误区 3：HRT 一律危险 → 错把晚启动人群结论套到所有人。
本周一试 + 思考题
THIS WEEK
若有持续疲劳、性欲下降、肌力流失，把"激素"列入下次体检（男性查清晨睾酮组合）。思考题：在用药前，你能先优化哪三个抬升内源激素的杠杆（睡眠/体脂/训练）？
SUB · 盆底 / 运动康复
RCT + Cochrane
盆底健康：被忽视的核心肌群
Pelvic Floor — The Overlooked Core
一句话结论
盆底肌是承托膀胱、子宫/前列腺、直肠的"吊床"，男女都有。它的松弛或过紧导致漏尿、脱垂、性功能下降——而正确的盆底肌训练可逆转，是 A 级证据。
科学背景 + 机制
盆底以肛提肌群为主，参与控尿控便、支撑盆腔器官、性高潮收缩。妊娠分娩、年龄、肥胖、慢性咳嗽/便秘、长期腹压会削弱它；久坐与焦虑则让部分人盆底过度紧张（高张型）——同样引起疼痛、尿急与性交痛。Cochrane 系统评价（Dumoulin 2018）：盆底肌训练（PFMT）使压力性尿失禁女性"治愈或改善"的可能性提高数倍，是一线疗法。男性前列腺术后 PFMT 同样加速控尿恢复。
可执行协议
• 先找对肌肉：想象"忍住排尿/排气"上提的感觉，不要同时收紧臀、腹、大腿或憋气。
• 剂量：每天 3 组，每组 8–12 次收缩；每次慢收上提 6–8 秒 + 等长放松同等时长，外加几次快速收缩。
• 持续：至少坚持 12 周见效，之后维持。
• 升级：若漏尿/脱垂/性交痛顽固，找盆底物理治疗师做评估（含是否高张需放松而非强化）。
女性特别提示 + 常见误区
女性视角：约 1/3 女性经历压力性尿失禁，产后与围绝经期是高峰；雌激素下降使尿道/阴道黏膜萎缩，可叠加局部雌激素改善。不是所有问题都是"太弱"——产后或慢性盆腔痛常是高张型，盲目猛练 Kegel 反而加重。
误区 1：漏尿是衰老必然 → 可治疗，别默默忍受护垫。
误区 2：Kegel 越多越好 → 高张型需要的是放松与呼吸，不是更多收缩。
误区 3：盆底只关乎女性 → 男性控尿、勃起硬度同样依赖它。
本周一试 + 思考题
THIS WEEK
用"等红灯/刷牙"作触发点，每天做 3 组规范盆底收缩，确认没有屏气和臀腿代偿。思考题：你分得清自己是"太弱"还是"太紧"吗？若练了不改善，是否该换方向？
SUB · 性功能 / 心血管
前瞻队列 + meta
性功能：心血管的金丝雀
Sexual Function — The Cardiovascular Canary
一句话结论
勃起功能障碍（ED）常是冠心病的早期预警：阴茎动脉更细，内皮功能障碍先在此显形，平均早于心脏事件 3–5 年。ED 是一个该查血管的"全身信号"，不只是卧室问题。
科学背景 + 机制
勃起依赖血管内皮释放一氧化氮（NO）使海绵体动脉舒张充血。阴茎动脉直径仅 1–2 mm，冠状动脉 3–4 mm——同样的内皮病变、动脉粥样硬化，细管先堵，于是 ED 早于心绞痛出现（"血管尺寸假说"）。Vlachopoulos 等 meta 分析显示 ED 显著预测未来心血管事件与全因死亡。女性性唤起同样是血管充血过程，性功能障碍与心血管风险也相关，只是研究较少。
可执行协议
• 把 ED 当红旗：新发持续 ED → 查血压、血脂、空腹血糖/HbA1c、腰围，评估十年心血管风险。
• 治根：戒烟、减脂、规律有氧 + 抗阻、控糖控压——同一套护血管方案同时改善勃起。
• 药物：PDE5 抑制剂（西地那非等）一线有效；硝酸酯类用药者禁用合用（严重低血压）。
• 女性：围绝经期性交痛/干涩首选局部阴道雌激素或润滑剂，安全有效。
女性特别提示 + 常见误区
女性视角：绝经泌尿生殖综合征（GSM）——阴道干涩、灼痛、反复尿路感染——很常见却长期被沉默。局部低剂量阴道雌激素全身吸收极少，连多数乳腺癌史者也可在医生评估下使用（NAMS 立场）。性欲是激素、血管、心理、关系的多因素结果，别只归咎"激素"。
误区 1：ED 纯属心理 → 40 岁后多数有血管/代谢成分，需体检。
误区 2：PDE5 药伤心 → 对多数心血管病情稳定者安全，反映的是血管获益；关键是别与硝酸酯同服。
误区 3：性问题是衰老认命 → 多数可治，且是体检线索。
本周一试 + 思考题
THIS WEEK
把"性功能变化"纳入你对心血管健康的自检清单——若有新发持续问题，预约一次代谢+血管评估而非默默吃药。思考题：你会把性功能当作一个客观的健康仪表，还是羞于提及的隐私？
SUB · 亲密关系 / 社会联结
前瞻队列 + meta
亲密关系：长寿的硬指标
Intimacy & Bonds — A Hard Longevity Metric
一句话结论
社会联结与亲密关系是可量化的长寿因子：孤独的死亡风险堪比每天吸 15 支烟。性活动与情绪、心血管、免疫正相关——但要诚实分清相关 ≠ 因果。
科学背景 + 机制
Holt-Lunstad 等 meta 分析（2010, 涵盖 30 万人）显示强社会联结使生存率提高约 50%，效应量与戒烟相当、超过运动和肥胖。机制：亲密接触与高潮释放催产素与内啡肽，降低皮质醇、血压与炎症标志；稳定关系还提供行为约束（更好的睡眠、饮食、就医）。性活动频率与较低心血管事件相关（如 Hall 2010, MMAS 队列），但活跃者往往本就更健康——因果方向需谨慎。
可执行协议
• 把关系当健康投资：像安排运动一样安排固定的高质量连接时间（伴侣、亲友）。
• 身体接触：每日数分钟拥抱/亲密接触即可触发催产素通路。
• 主动抗孤独：识别孤独信号，定期面对面社交（线上替代不了）。
• 沟通先于技巧：性满意度更多取决于关系质量与开放沟通，而非频率指标。
女性特别提示 + 常见误区
女性视角：围绝经期性欲变化是多因素的——激素只是其一，睡眠差、情绪、关系满意度、伴侣健康、自我形象往往权重更大。把所有问题归给"雌激素低"会错过更可改变的杠杆。女性性反应更受情境与情感驱动，沟通本身即是有效"干预"。
误区 1：性与亲密是年轻人的事 → 联结需求与活动贯穿全生命周期。
误区 2：频率 = 满意度 → 质量、安全感、沟通才是关键变量。
误区 3：性活动直接"延寿" → 多为相关，背后是健康与关系质量的共同因。
本周一试 + 思考题
THIS WEEK
本周安排一次无手机、不赶时间的高质量连接（深谈、共同散步或拥抱），把它当成和运动同等的健康习惯。思考题：在你的长寿清单里，"关系"和"血脂"哪个被严肃对待得更少？
深入思考
把睾酮/雌激素当"抗衰开关"——边界在哪？
激素治疗是对确诊缺乏的替代，把指标拉回正常区间能恢复对应功能；但对本就正常者，外源激素并不"增寿"，反会抑制内源轴、带来红细胞增多、血栓等风险。关键区分"纠正缺乏"与"超生理增强"。女性 MHT 的获益—风险高度依赖启动时机与个体史，应个体化而非一刀切。
ED 作为心血管预警，假阳性与焦虑成本如何权衡？
把 ED 视为信号确实能提前发现代谢/血管病——这是低成本高价值的筛查触发点。但并非所有 ED 都源于血管（药物、心理、睡眠、关系皆可致），过度医学化也会制造焦虑。合理做法是：新发、持续、与年龄不符的 ED 触发一次系统评估，而非每次都恐慌。
盆底"练得越多越好"为什么是危险的直觉？
肌肉的功能是收放自如，不是恒定收紧。高张型盆底（常见于慢性盆腔痛、产后、长期紧张）的问题是不会放松，盲目加练 Kegel 会加重疼痛、尿急与性交痛。先辨"弱"还是"紧"，必要时用反向训练（放松、呼吸、生物反馈）。这也是为何专业评估优于自行猛练。
"社会联结=长寿"的因果链有多牢？
联结与死亡率的关联在多个大队列中稳健、效应量大，且有催产素/皮质醇/行为约束等合理机制。但仍是观察性证据：健康者更易维持关系（反向因果），社会经济地位是潜在混杂。结论可行动（投资关系几乎无害且大概率有益），但别把它当成单一干预的因果保证。羞耻与回避还让人把可治疗的问题（漏尿、ED、GSM、性交痛）当成"认命"，去羞耻化、仪表盘化本身就是被低估的健康升级。
